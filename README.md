## Descripción

**Gendiff** es una herramienta de línea de comandos que compara archivos de configuración en formato **JSON** o **YAML** y muestra las diferencias de forma estructurada.

---

## Instalación

Clona este repositorio y ejecuta:

```bash
npm install
```
